Install

Install the operator from the Helm repo (images pull from ghcr.io/ianp94/basquin-*):

$ helm repo add basquin https://ianp94.github.io/basquin/charts
$ helm repo update
$ helm install basquin basquin/basquin-operator --namespace basquin-system --create-namespace --set fullnameOverride=basquin

Or grab a basquin CLI binary (linux / macOS / Windows, amd64 + arm64) from the release assets above, then basquin instrument / basquin run — see the operator guide.
